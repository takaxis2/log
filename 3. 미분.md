
## 요약

본 문서는 데이터 과학과 인공지능 분야의 기초가 되는 미분의 핵심 원리를 종합적으로 설명한다. 함수의 극한 개념에서 시작하여 수렴과 발산, 연속성의 조건을 정의하고, 이를 바탕으로 미분의 핵심인 평균변화율과 순간변화율(미분계수)의 개념을 도출한다. 또한, 다항함수를 미분하는 데 필요한 주요 미분법 공식을 체계적으로 정리하고, 연속성과 미분 가능성의 관계를 명확히 한다. 마지막으로, 이러한 수학적 개념들을 파이썬의 `SymPy`, `NumPy`, `matplotlib` 라이브러리를 통해 실제로 구현하는 방법을 구체적인 함수와 클래스를 통해 제시하여 이론과 실제의 간극을 연결하는 것을 목표로 한다.

## I. 함수의 극한

극한은 미분 개념을 이해하기 위한 근본적인 토대이다. 함수가 특정 지점에서 어떤 값에 가까워지는지를 분석하며, 이는 수렴, 발산, 연속성 등의 개념으로 이어진다.

### A. 수렴과 발산

- **수렴 (Convergence)**: 변수 `x`가 특정 값 `a`에 한없이 가까워질 때, 함수 `f(x)`의 값이 특정 값 `b`에 한없이 가까워지는 현상. 이때 `b`를 함수 `f(x)`의 극한값 또는 극한이라 부른다. 극한은 함수가 수렴할 때 존재한다.
    - **표현**: `x → a`일 때 `f(x) → b` 또는 `lim(x→a) f(x) = b`
    - `**x → a**`**의 의미**: `x`는 `a`와 같지 않으면서(`x ≠ a`) `a`에 한없이 가까워지는 상태를 의미하며, 좌극한(`x → a-0`)과 우극한(`x → a+0`)을 모두 포함한다.
    - **무한대에서의 극한**: `x`가 양의 무한대(`∞`) 또는 음의 무한대(`-∞`)로 갈 때도 함수가 특정 값에 수렴할 수 있다.
- **발산 (Divergence)**: 변수 `x`가 `a`에 가까워질 때 `f(x)`가 특정 실수 값으로 수렴하지 않는 상태. 발산하는 경우 극한값은 존재하지 않는다.
    - **양의 무한대 발산**: `f(x)`가 한없이 커지는 경우 (`lim(x→a) f(x) = ∞`)
    - **음의 무한대 발산**: `f(x)`가 한없이 작아지는 경우 (`lim(x→a) f(x) = -∞`)
    - **진동**: `f(x)`가 특정 값으로 수렴하지 않고 일정한 범위 내에서 계속해서 변동하는 경우

### B. 극한값의 존재 조건

함수의 극한값이 특정 지점에서 존재하기 위해서는 좌극한과 우극한이 모두 존재하며 그 값이 서로 일치해야 한다.

- **좌극한**: `x`가 `a`보다 작은 쪽에서 `a`로 접근할 때 `f(x)`가 가까워지는 값.
- **우극한**: `x`가 `a`보다 큰 쪽에서 `a`로 접근할 때 `f(x)`가 가까워지는 값.
- **존재 조건**: `lim(x→a+) f(x) = lim(x→a-) f(x) = b`

### C. 함수의 연속성

함수의 그래프가 특정 구간에서 끊어지지 않고 연결되어 있는 상태를 연속이라고 한다. 한 지점에서 연속이 되기 위한 조건은 다음과 같다.

1. **함수값의 존재**: `x = a`에서 함수 `f(x)`가 정의되어 있어야 한다 (`f(a)`가 존재).
2. **극한값의 존재**: `x → a`일 때의 극한값 `lim(x→a) f(x)`가 존재해야 한다 (즉, 좌극한 = 우극한).
3. **함수값과 극한값의 일치**: 극한값과 함수값이 같아야 한다 (`lim(x→a) f(x) = f(a)`).

이 세 가지 조건 중 하나라도 만족하지 않으면 함수는 `x = a`에서 불연속이다.

## II. 미분의 개념과 계산

미분은 함수의 특정 지점에서의 '순간적인 변화율'을 측정하는 도구로, 함수의 증가와 감소, 극대점과 극소점을 파악하는 데 결정적인 역할을 한다.

### A. 변화율: 평균과 순간

- **평균변화율 (Average Rate of Change)**: 함수 `y = f(x)`에서 `x`값의 증가량에 대한 `y`값의 증가량의 비율을 의미한다.
    - **수학적 표현**: `Δy / Δx` (여기서 `Δ`는 변화량을 의미)
    - **기하학적 의미**: 그래프 위의 두 점을 잇는 직선의 기울기.
    - **계산식**:
        - `(f(b) - f(a)) / (b - a)`
        - `(f(a + h) - f(a)) / h` (여기서 `h = b - a`)
- **순간변화율 (Instantaneous Rate of Change)**: 평균변화율에서 `x`의 증가량(`Δx` 또는 `h`)을 0에 한없이 가깝게 보낼 때의 극한값. 이를 **미분계수**라고도 한다.
    - **수학적 표현**: `f'(a) = lim(h→0) (f(a + h) - f(a)) / h`
    - **기하학적 의미**: `x = a`인 지점에서의 접선의 기울기.
    - **미분계수의 부호**:
        - `f'(a) > 0` 이면 함수 `f(x)`는 `x = a`에서 증가 상태이다.
        - `f'(a) < 0` 이면 함수 `f(x)`는 `x = a`에서 감소 상태이다.

### B. 도함수와 미분법

- **도함수 (Derivative Function)**: 함수 `f(x)`를 미분하여 얻은 새로운 함수 `f'(x)`. 임의의 점 `x`에서의 미분계수(순간변화율)를 나타내는 함수로, `f'(1)`, `f'(2)` 등을 개별적으로 계산하는 번거로움을 줄여준다.
    - **다양한 표기법**: `y'`, `f'(x)`, `dy/dx`, `d/dx f(x)`
- **주요 미분법 (Differentiation Rules)**: 복잡한 함수를 효율적으로 미분하기 위한 공식들이다.

|   |   |   |
|---|---|---|
|구분|함수 형태 `f(x)`|도함수 `f'(x)`|
|**상수**|`c`|`0`|
|**거듭제곱**|`xⁿ`|`nxⁿ⁻¹`|
|**로그**|`log(x)`|`1/x`|
|**지수**|`eˣ`|`eˣ`|
|**선형조합**|`c * g(x) ± h(x)`|`c * g'(x) ± h'(x)`|
|**곱셈법칙**|`g(x) * h(x)`|`g'(x)h(x) + g(x)h'(x)`|
|**연쇄법칙**|`h(g(x))`|`dy/dx = (dy/dg) * (dg/dx)`|

### C. 미분 가능성

함수가 특정 지점에서 미분 가능하다는 것은 해당 지점에서 미분계수가 존재한다는 의미이다. 미분 가능성은 연속성과 밀접한 관계를 가진다.

- **관계**: 함수가 특정 점에서 **미분 가능하면 반드시 연속**이다.
- **역의 불성립**: 그러나 함수가 특정 점에서 **연속이라고 해서 반드시 미분 가능한 것은 아니다**.
- **예시**: 함수 `y = |x|`는 `x = 0`에서 그래프가 끊어지지 않아 연속이지만, 뾰족점을 가지므로 좌미분계수와 우미분계수가 달라 미분은 불가능하다.

## III. 파이썬을 활용한 미분 구현

파이썬의 과학 계산 라이브러리를 사용하면 미분의 개념을 코드로 구현하고 시각화할 수 있다.

### A. 주요 라이브러리

- **SymPy**: 기호 연산(Symbolic Mathematics)을 위한 라이브러리로, 극한 계산, 인수분해, 미분 등 복잡한 수학 공식을 다루는 데 사용된다.
- **NumPy**: 수치 계산을 위한 핵심 라이브러리로, 배열 연산 및 수치 데이터 처리에 사용된다. 소스에서는 `Derivative()` 클래스가 `NumPy` 라이브러리의 일부로 언급되었다.
- **Matplotlib**: 데이터 시각화를 위한 라이브러리로, 함수 그래프를 그리는 데 사용된다.
