
## 개요

본 문서는 데이터 과학과 인공지능 분야의 근간이 되는 선형대수학의 핵심 개념을 요약한다. 데이터의 기본 단위인 스칼라에서부터 벡터, 행렬, 텐서로 확장되는 데이터 구조의 계층을 설명하고, 벡터의 기하학적·대수적 정의와 주요 특성을 상세히 다룬다. 특히, 벡터 공간의 구조를 이해하는 데 필수적인 선형 결합, 선형 독립과 종속, 그리고 부분공간의 기저와 랭크 개념을 체계적으로 정리한다. 또한, 범주형 변수를 수치 데이터로 변환하는 '가변수(dummy variable)' 활용 사례를 통해 이러한 수학적 개념이 인공지능 모델에서 어떻게 실용적으로 적용되는지 제시하며, 각 개념을 Python 코드로 구현하는 데 사용되는 주요 라이브러리 함수를 명시한다.

## 1. 데이터 구조의 계층: 스칼라, 벡터, 행렬, 텐서

데이터 과학에서 사용되는 데이터는 차원에 따라 스칼라, 벡터, 행렬, 텐서로 구분된다. 이들은 차원이 확장되는 계층적 구조를 이룬다.

- **스칼라 (Scalar):** 단일 숫자로 구성된 데이터. 0차원 텐서로 간주할 수 있다. 실수 집합 R의 원소로 표현된다.
- **벡터 (Vector):** 여러 숫자가 특정 순서대로 배열된 데이터. 크기와 방향을 가지며, 1차원 텐서에 해당한다.
- **행렬 (Matrix):** 벡터가 여러 개 모여 행과 열의 2차원 구조를 이룬 데이터. 2차원 텐서로 볼 수 있다.
- **텐서 (Tensor):** 행렬이 여러 개 결합된 3차원 이상의 데이터 구조. n차원 배열로 일반화된다.

|   |   |   |   |
|---|---|---|---|
|데이터 구조|차원 (텐서 기준)|설명|예시|
|**스칼라**|0차원|하나의 숫자|96 (평균 급여)|
|**벡터**|1차원|숫자의 순서 있는 배열|`[0.48, 3, 96, 3]` (한 사원의 데이터)|
|**행렬**|2차원|벡터의 집합 (행과 열)|HR 데이터 테이블, 그레이스케일 이미지|
|**텐서**|3차원 이상|행렬의 집합|RGB 컬러 이미지 ('R', 'G', 'B' 채널별 2차원 행렬)|

## 2. 벡터의 핵심 속성 및 종류

벡터는 데이터 포인트, 특성, 방향 등 다양한 정보를 표현하는 데 사용되는 기본 단위이다.

### 2.1. 정의와 특징

- **기하학적 정의:** 크기(길이)와 방향을 가진 물리량.
- **대수적 정의:** 실수 또는 함수를 성분으로 갖는 순서쌍 또는 배열.
- **주요 특징:**
    - 벡터는 크기뿐만 아니라 방향도 중요한 요소이다.
    - 크기와 방향이 동일하면 서로 다른 위치에 있어도 같은 벡터로 간주된다.
    - Python에서는 `numpy.array()` 함수를 사용하여 벡터를 생성한다.

### 2.2. 특수 벡터

특정 목적을 위해 정의된 벡터들은 선형대수 연산에서 중요한 역할을 한다.

- **단위 벡터 (Unit Vector):**
    - **정의:** 길이가 1인 벡터. 벡터 기호 위에 햇(^)을 붙여 `v̂`와 같이 표기한다.
    - **계산:** 원래 벡터를 자신의 크기(노름, Norm)로 나누어 구한다. 벡터 `v = [x, y]`의 노름 `||v||`은 `√(x² + y²)`로 계산된다.
    - **Python 구현:** `scipy.linalg.norm()` 함수로 벡터의 노름을 계산할 수 있다.
- **위치 벡터 (Position Vector):**
    - **정의:** 좌표계의 원점(0,0)을 시작점으로 하도록 재표현한 벡터. 벡터의 끝점 좌표가 해당 벡터의 성분이 된다.
- **영 벡터 (Zero Vector):**
    - **정의:** 모든 성분이 0으로 구성된 벡터. 크기가 0이며 시작점과 끝점이 같다.
    - **특징:**
        - 모든 벡터 공간은 선형 결합을 위해 반드시 영 벡터를 포함해야 한다.
        - 덧셈에 대한 항등원 역할을 한다 (`x + 0 = x`).
        - 모든 벡터와 평행하면서 동시에 수직인 유일한 벡터이다.
    - **Python 구현:** `numpy.zeros()` 함수로 생성한다.

### 2.3. 인공지능에서의 활용

벡터는 인공지능 모델이 이해할 수 있도록 데이터를 변환하는 데 핵심적으로 사용된다. 대표적인 예로 범주형 변수를 수치 벡터로 변환하는 **가변수(dummy variable) 혹은 원-핫 인코딩(one-hot encoding)**이 있다. 예를 들어 '연봉'이라는 변수가 'Low', 'Medium', 'High' 세 가지 범주를 가질 때, 이를 다음과 같은 벡터로 표현할 수 있다.

- Low: `[1, 0, 0]`
- Medium: `[0, 1, 0]`
- High: `[0, 0, 1]`

## 3. 벡터 간의 관계: 선형 결합과 독립성

벡터들 간의 관계를 정의하는 선형 결합과 독립성은 벡터 공간의 구조를 파악하는 기반이 된다.

### 3.1. 선형 결합과 생성(Span)

- **선형 결합 (Linear Combination):** 주어진 벡터들에 각각 스칼라를 곱한 후 더하여 새로운 벡터를 만드는 연산. 예를 들어, 두 벡터 `a`와 `b`가 있을 때, `c₁a + c₂b`는 `a`와 `b`의 선형 결합이다.
- **생성 (Span):** 주어진 벡터들의 모든 가능한 선형 결합으로 만들 수 있는 벡터들의 집합. 예를 들어, `span(a, b)`는 벡터 `a`와 `b`로 생성할 수 있는 모든 벡터의 공간을 의미한다.

### 3.2. 선형 독립과 종속

- **선형 독립 (Linearly Independent):** 벡터 집합 `{a₁, a₂, ..., aₙ}`에 대해, 선형 결합 `c₁a₁ + c₂a₂ + ... + cₙaₙ = 0`을 만족시키는 유일한 해가 모든 스칼라 계수 `cᵢ`가 0인 경우(`c₁ = c₂ = ... = cₙ = 0`)를 의미한다. 이는 집합 내 어떤 벡터도 다른 벡터들의 선형 결합으로 표현될 수 없음을 뜻한다.
- **선형 종속 (Linearly Dependent):** 위 선형 결합 식에서 모든 계수가 0이 아니어도 등식을 만족하는 해가 하나 이상 존재하는 경우이다. 이는 집합 내 최소 하나 이상의 벡터가 다른 벡터들의 선형 결합으로 표현될 수 있음을 의미한다.

## 4. 벡터 공간의 구조: 기저와 랭크

벡터 공간은 벡터들이 존재하며 연산이 정의되는 추상적인 공간이다. 기저와 랭크는 이 공간의 차원과 구조를 설명하는 핵심 지표이다.

### 4.1. 벡터 공간과 부분 공간

- **벡터 공간 (Vector Space):** 선형 결합 연산에 대해 닫혀 있는 벡터들의 집합. 즉, 공간 내의 어떤 벡터들을 가져와 선형 결합을 해도 그 결과가 항상 같은 공간 내에 존재해야 한다.
- **부분 공간 (Subspace):** 더 큰 벡터 공간 `Rⁿ` 내에 존재하면서, 그 자체로 벡터 공간의 조건을 만족하는 부분적인 공간 `V`. 부분 공간이 되기 위한 세 가지 조건은 다음과 같다.
    1. 영 벡터(0)를 포함해야 한다.
    2. 공간 내의 모든 벡터 `w`에 대해, 임의의 스칼라 `c`를 곱한 `cw`도 공간에 포함되어야 한다 (스칼라 곱셈에 대해 닫혀 있음).
    3. 공간 내의 두 벡터 `a`, `b`의 합 `a + b`도 공간에 포함되어야 한다 (덧셈에 대해 닫혀 있음).

### 4.2. 기저 (Basis)

- **정의:** 특정 벡터 공간을 생성(span)하는 데 필요한 최소한의, 선형 독립적인 벡터들의 집합.
- **특징:**
    - 기저 벡터들의 선형 결합으로 해당 공간의 모든 벡터를 유일하게 표현할 수 있다.
    - 공간의 차원(dimension)은 기저를 구성하는 벡터의 개수와 같다. (예: 2차원 공간은 2개의 기저 벡터, 3차원 공간은 3개의 기저 벡터가 필요).
- **표준 기저 벡터 (Standard Basis Vector):** 기저 벡터 중에서 하나의 원소만 1이고 나머지 원소는 모두 0인 벡터. 예를 들어, 3차원 공간의 표준 기저는 `[1, 0, 0]`, `[0, 1, 0]`, `[0, 0, 1]`이다.

### 4.3. 랭크 (Rank)

- **정의:** 행렬에서 선형 독립인 행 또는 열의 최대 개수. 행렬이 나타내는 벡터 공간의 실질적인 차원을 의미한다.
- **특징:** 행렬 `A`의 랭크와 그 전치행렬 `Aᵀ`의 랭크는 항상 같다. (`rank(A) = rank(Aᵀ)`)
- **Python 구현:** `numpy.linalg.matrix_rank()` 함수를 사용하여 계산한다.

## 5. Python을 활용한 구현 요약

선형대수학의 개념들은 Python의 `numpy`와 `scipy` 라이브러리를 통해 효율적으로 구현될 수 있다.

|   |   |   |
|---|---|---|
|개념|Python 함수|라이브러리|
|벡터 및 행렬 생성|`numpy.array()`|`numpy`|
|벡터의 크기 (노름)|`linalg.norm()`|`scipy`|
|영 벡터 생성|`numpy.zeros()`|`numpy`|
|행렬의 랭크|`linalg.matrix_rank()`|`numpy`|
