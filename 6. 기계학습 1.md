### 1. 기계학습 (Machine Learning)의 정의 및 특징

- **정의:** 컴퓨터가 데이터(경험 $E$)로부터 특정 문제 해결을 위한 지식을 자동으로 추출하여, 특정 작업($T$)에 대한 성능($P$)이 향상되도록 시스템의 구조나 파라미터를 수정해 나가는 기술입니다.
    
- **핵심:** 데이터에서 패턴과 지식을 스스로 학습하여 향후 유사한 작업을 더 효율적으로 처리하는 데 목적이 있습니다.
    

### 2. 기계학습의 분류

데이터의 형태와 학습 방식에 따라 크게 세 가지로 분류됩니다.

|**분류**|**특징**|**주요 과제 및 예시**|
|---|---|---|
|**지도학습 (Supervised)**|입력값과 그에 대한 **정답(Label)**이 주어진 데이터를 학습|분류(Classification), 회귀(Regression)|
|**비지도학습 (Unsupervised)**|정답 없이 입력 데이터만으로 **패턴이나 구조**를 스스로 발견|군집화(Clustering), 차원 축소, 이상치 탐지|
|**강화학습 (Reinforcement)**|에이전트가 환경과 상호작용하며 **보상(Reward)**을 극대화하는 방향으로 학습|게임 AI(알파고), 로봇 제어|

### 3. 기계학습의 주요 대상 문제

- **분류 (Classification):** 입력 데이터를 정해진 몇 개의 범주(Class) 중 하나로 나누는 문제입니다. (예: 스팸 메일 판별, 필기 문자 인식)
    
- **회귀 (Regression):** 입력 변수를 바탕으로 연속적인 수치(실수값)를 결과로 예측하는 문제입니다. (예: 주택 가격 예측, 주가 예측)
    
- **군집화 (Clustering):** 명확한 기준(정답) 없이 데이터 간의 유사도를 측정하여 비슷한 것들끼리 그룹으로 묶는 문제입니다.
    

### 4. 모델의 학습과 평가 기법

- **데이터 분할:** 전체 데이터를 모델 학습을 위한 **훈련 데이터(Training data)**와 성능 측정을 위한 **테스트 데이터(Test data)**로 분리하여 사용합니다.
    
- **K-폴드 교차 검증 (K-fold Cross Validation):** 테스트 데이터가 부족할 때 모델의 일반화 성능을 객관적으로 평가하기 위해 데이터를 K개로 나누어 번갈아 가며 테스트하는 기법입니다.
    
- **성능 평가 지표:** 정확도(Accuracy), 정밀도(Precision), 재현율(Recall) 등을 통해 모델의 성능을 정량적으로 측정합니다.
    

### 5. 기타 주요 개념 및 활용

- **과적합 (Overfitting):** 모델이 학습 데이터에 너무 과하게 최적화되어, 새로운 데이터(테스트 데이터)에 대한 예측 성능이 떨어지는 현상입니다.
    
- **추천 시스템 (Filtering):**
    
    - **콘텐츠 기반 필터링:** 사용자가 과거에 선호했던 아이템과 유사한 특성을 가진 아이템을 추천합니다.
        
    - **협업 필터링:** 유사한 성향을 가진 다른 사용자들의 데이터를 기반으로 아이템을 추천합니다.
