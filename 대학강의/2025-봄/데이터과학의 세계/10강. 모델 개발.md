
## Executive Summary

본 문서는 데이터 과학에서의 모델 개발 프로세스를 종합적으로 분석하고 핵심 개념을 정리합니다. 모델은 하나 이상의 독립 변수(예측 변수)를 기반으로 종속 변수(목표 변수)를 예측하는 수학적 방정식으로 정의됩니다. 모델의 정확성은 데이터의 양과 질, 그리고 적절한 모델 타입 선택에 크게 좌우됩니다.

핵심 모델 유형으로는 하나의 예측 변수를 사용하는 **단순 선형 회귀(SLR)**, 둘 이상의 예측 변수를 활용하는 **다중 선형 회귀(MLR)**, 그리고 변수 간 비선형 관계를 모델링하는 **다항 회귀**가 있습니다. 다항 회귀는 변수의 고차항을 사용하지만, 회귀 계수(파라미터)가 선형 결합으로 표현되므로 일반 선형 회귀의 특별한 형태로 간주됩니다.

모델 평가는 **시각화**와 **정량적 측정** 두 가지 축으로 이루어집니다. SLR의 경우 회귀도와 잔차도를 통해 데이터 적합도를 시각적으로 평가하며, MLR은 분포도를 사용합니다. 정량적 평가 지표로는 모델의 설명력을 나타내는 **결정계수(R²)**와 예측 오차의 크기를 측정하는 **평균제곱오차(MSE)**가 핵심적으로 사용됩니다. R²는 1에 가까울수록, MSE는 0에 가까울수록 모델의 성능이 우수함을 의미합니다.

최적의 모델을 선택하는 과정은 단일 지표에 의존하지 않고, 예측값의 합리성, 시각적 평가, 정량적 수치를 종합적으로 고려하는 의사결정이 필요합니다. 예를 들어, 예측 변수가 추가된 MLR 모델은 SLR 모델보다 MSE가 낮아지는 경향이 있지만, 이것이 항상 더 나은 모델임을 보장하지는 않으며 과적합 등의 문제를 함께 고려해야 합니다.

## 1. 모델의 기본 개념

### 1.1. 모델의 정의와 목적

모델은 주어진 데이터를 기반으로 특정 목표 값을 예측하는 데 사용되는 **수학적 방정식(Mathematical Equation)**으로 정의됩니다. 이는 한 개 이상의 예측 변수(독립 변수)와 목표 변수(종속 변수) 간의 연관성을 수학적으로 구조화한 형태입니다.

- **예측 변수 (독립 변수 또는 피처):** 예측의 기반이 되는 입력 값입니다. (예: 중고차 가격 예측 시 '연비(Highway-mpg)')
- **목표 변수 (종속 변수):** 모델을 통해 예측하고자 하는 값입니다. (예: '가격(Price)')

### 1.2. 모델 성능 향상 요인

모델의 예측 정확도를 높이기 위한 핵심 요인은 다음과 같습니다.

1. **데이터의 양과 관련성:**
    - 관련성 높은 데이터를 많이 보유할수록 모델의 정확도는 향상됩니다. 예를 들어, 중고차 가격 예측 시 '연비', '차체 중량', '엔진 크기' 등 다양한 피처를 추가하면 더 정확한 예측이 가능합니다.
    - 모델이 학습하지 않은 중요한 변수(예: 자동차 '색상')가 누락되면, 동일한 조건의 다른 차량에 대해 잘못된 예측을 할 수 있습니다.
2. **다양한 모델 타입 시도:**
    - 데이터의 특성과 변수 간 관계에 따라 가장 적합한 모델을 선택해야 합니다. 주요 회귀 모델 유형은 다음과 같습니다.
        - **단순 선형 회귀 (SLR):** 예측 변수가 1개인 경우 사용됩니다.
        - **다중 선형 회귀 (MLR):** 예측 변수가 2개 이상인 경우 사용됩니다.
        - **다항 회귀:** 예측 변수와 목표 변수 간의 관계가 선형이 아닌 곡선 형태일 때 사용됩니다. 예측 변수를 다항식(2차 이상)으로 구성합니다.

## 2. 회귀 모델의 종류

### 2.1. 단순 선형 회귀 (Simple Linear Regression, SLR)

단순 선형 회귀는 데이터 포인트들을 가장 잘 표현하는 하나의 직선(최소제곱선, Least Square Line)을 찾는 모델링 기법입니다. 이는 실제 값과 예측 값의 차이인 **잔차(Residual) 제곱의 합**을 최소화하는 방식으로 이루어집니다.

- **SLR 함수:** `Y = β₀ + β₁x + u`
    - `Y`: 예측하려는 목표 값
    - `x`: 예측 변수
    - `β₀`: y 절편 (직선이 Y축과 만나는 지점)
    - `β₁`: 기울기 (x가 1 증가할 때 Y의 변화량)
    - `u`: 잔차 또는 노이즈(오차)

### 2.2. 다중 선형 회귀 (Multiple Linear Regression, MLR)

다중 선형 회귀는 n개의 피처(독립 변수)를 사용하여 목표 변수를 예측하는 모델입니다. 이는 단순 선형 회귀를 확장하여 여러 차원의 데이터를 설명하는 선을 탐색합니다.

- **MLR 함수 (예: n=2):** `Y = β₀ + β₁x₁ + β₂x₂ + ...`
    - `Y`: 반응 변수 (종속 변수)
    - `x₁, x₂, ...`: 예측 변수 (독립 변수, 피처)
    - `β₀, β₁, ...`: 계수 (Coefficient)
    - `Error`: 랜덤 오차 또는 노이즈

## 3. 다항 회귀와 데이터 파이프라인

### 3.1. 개념 및 활용

다항 회귀는 예측(독립) 변수가 다항식(예: 2차항, 3차항)으로 구성된 회귀 모델입니다. 이는 독립 변수와 종속 변수가 선형 관계가 아닌 **곡선 형태의 관계**를 가질 때 유용합니다. 다항 회귀는 독립 변수에 지수승을 적용하여 새로운 변수를 생성한 뒤, 이를 다중 선형 회귀 모델에 적용하는 방식이므로 **일반 선형 회귀 모델의 특별한 형태**로 간주됩니다.

- **선형 회귀 모델:** 회귀 계수(파라미터)를 선형 결합으로 표현할 수 있는 모델. 다항 회귀는 여기에 해당합니다.
- **비선형 회귀 모델:** 파라미터를 선형 결합식으로 표현할 수 없는 모델.

### 3.2. 모델의 종류

- **일변수(일차원) 다항 회귀:**
    - 2차(Quadratic): `Ŷ = β₀ + β₁x + β₂x²`
    - 3차(Cubic): `Ŷ = β₀ + β₁x + β₂x² + β₃x³`
- **다변수(다차원) 다항 회귀:**
    - 두 개 이상의 예측 변수를 다항식으로 조합합니다. 예를 들어, 두 변수(X₁, X₂)를 가진 2차 다항식은 `X₁`, `X₂`, `X₁X₂`, `X₁²`, `X₂²` 등의 항을 포함할 수 있습니다.

### 3.3. 데이터 처리 단순화를 위한 파이프라인

복잡한 데이터 처리 단계를 단순화하기 위해 파이프라인을 구축할 수 있습니다. 이는 여러 변환 과정을 순차적으로 묶어 한 번에 실행하도록 합니다.

- **일반적인 파이프라인 단계:**
    1. **정규화 (Normalization):** 모든 피처가 동일한 스케일(중요도)을 갖도록 조정합니다. (예: 최소-최대 정규화, Z-점수 표준화)
    2. **다항 변환 (Polynomial Transform):** 기존 피처를 다항 피처로 변환합니다.
    3. **선형 회귀 (Linear Regression):** 변환된 피처를 사용하여 모델을 학습시킵니다.

## 4. 모델 평가 방법론

모델이 데이터에 얼마나 잘 적합(Fit)하는지 평가하는 것은 모델 개발의 필수적인 단계입니다. 평가는 시각적 방법과 정량적 방법을 병행합니다.

### 4.1. 시각화를 통한 평가

|   |   |   |
|---|---|---|
|모델 종류|시각화 방법|설명|
|**단순 선형 회귀 (SLR)**|**회귀도 (Regression Plot)**|• 두 변수 간 관계의 강도와 방향성(양/음의 상관관계)을 시각적으로 제공합니다.|
||**잔차도 (Residual Plot)**|• 잔차(실제값 - 예측값)를 시각화합니다.<br>• 잔차들이 x축 주변에 무작위로 흩어져 있다면 선형 모델이 적절함을 의미합니다.<br>• 특정 패턴이 보이면 비선형 모델을 고려해야 합니다.|
|**다중 선형 회귀 (MLR)**|**분포도 (Distribution Plot)**|• 예측 변수가 2개 이상이므로 회귀도나 잔차도로 시각화할 수 없습니다.<br>• 예측값과 실제값의 분포를 비교하여 모델의 적합도를 평가합니다.|

### 4.2. 정량적 평가 지표 (표본 내 평가)

시각화와 더불어 모델의 정확성을 수치로 확인하기 위한 정량적 측정치가 필요합니다.

1. **결정계수 (R-squared, R²)**
    - **정의:** 모델의 **설명력**을 나타내는 지표로, 데이터가 회귀선에 얼마나 가까운지를 측정합니다.
    - **해석:** 값의 범위는 0에서 1 사이이며, **1에 가까울수록** 모델이 데이터를 더 잘 설명하고 적합도가 높음을 의미합니다.
2. **평균제곱오차 (Mean Squared Error, MSE)**
    - **정의:** 오차(실제값 - 예측값) 제곱의 평균값입니다.
    - **공식:** `MSE = (1/n) * Σ(yᵢ - ŷᵢ)²`
    - **해석:** 예측값과 실제값의 차이를 측정하며, **0에 가까울수록** 모델의 오차가 작고 성능이 우수함을 의미합니다.

## 5. 예측과 의사결정

데이터셋에 가장 적합한 모델을 선택하는 것은 여러 요소를 종합적으로 고려하여 판단해야 합니다.

### 5.1. 종합적 판단 기준

- **예측 값의 합리성:** 모델이 생성한 예측 결과가 현실적으로 타당한지 검토합니다.
- **시각화 평가:** 회귀도, 잔차도, 분포도 등을 통해 모델의 적합성을 시각적으로 확인합니다.
- **평가용 수치 확인:** R²와 MSE 값을 비교하여 모델의 정량적 성능을 평가합니다.

### 5.2. 모델 비교 시 주요 고려사항

- **MSE와 모델 복잡성:**
    - 더 작은 MSE가 항상 더 좋은 모델을 의미하는 것은 아닙니다. 특히 과적합(overfitting)된 모델은 훈련 데이터에 대해서는 MSE가 매우 낮지만 새로운 데이터에 대해서는 성능이 저하될 수 있습니다.
- **SLR vs. MLR:**
    - 일반적으로 모델에 예측 변수가 추가되면(SLR → MLR) 모델 오차가 줄어들어 MLR의 MSE가 SLR의 MSE보다 작아지는 경향이 있습니다.
- **선형 회귀 vs. 다항 회귀:**
    - 데이터가 비선형 관계를 가질 경우, 다항 회귀 모델이 단순 선형 회귀 모델보다 더 작은 MSE를 가집니다.
- **R² 값:**
    - 여러 모델을 비교할 때, R² 값이 1에 더 가까운 모델이 데이터에 더 잘 적합한다고 판단할 수 있습니다.
