
## Executive Summary

본 문서는 세종사이버대학교 김명배 교수의 '데이터과학과 AI를 위한 파이썬' 과정 중 1강 '기초수학'의 핵심 내용을 요약 및 분석한 것이다. 이 자료는 데이터 과학과 인공지능 분야의 기반이 되는 필수적인 수학 개념을 다룬다. 주요 내용은 수학적 표기에 사용되는 그리스 문자의 이해, 변수와 상수를 포함한 수식의 기본 구조, 그리고 방정식과 부등식의 원리 및 특성으로 구성된다.

핵심적으로, 수학에서는 변수 표현을 위해 그리스 문자를 광범위하게 사용하며, 이는 AI 관련 논문 및 알고리즘 이해에 필수적이다. 수식은 변수, 상수, 항, 계수, 차수 등의 요소로 구성되며, 이들의 정의를 명확히 이해하는 것이 중요하다. 방정식은 미지수의 특정 값(해 또는 근)에 대해서만 참이 되는 등식이며, 항등식은 모든 값에 대해 참이 된다. 반면, 부등식은 등호 대신 부등호를 사용하여 수의 대소 관계를 나타내며, 특히 양변에 음수를 곱하거나 나눌 때 부등호의 방향이 바뀐다는 핵심적인 성질을 가진다.

## 2. 변수와 수식의 이해

수식은 변수와 상수를 연산자로 연결하여 표현한 것으로, 그 구조를 이해하는 것은 수학적 모델링의 기초가 된다.

### 2.1. 변수와 상수

- **변수(Variable)**: '변하는 수'를 의미하며, 상황에 따라 다른 값을 가질 수 있는 수를 지칭한다.
- **상수(Constant)**: 변하지 않는 고정된 수를 의미한다.

### 2.2. 파이썬 변수 명명 규칙

파이썬 환경에서 변수를 선언할 때는 다음 규칙을 준수해야 한다.

- 문자, 숫자, 밑줄 문자(_)를 사용할 수 있다.
- 공백은 사용할 수 없다.
- 대문자와 소문자를 구분한다.
- 숫자로 시작할 수 없다.
- 특수 문자(+, -, @, % 등)는 사용할 수 없다.

### 2.3. 수식의 구성 요소

- **항(Term)**: 숫자 또는 문자의 곱으로 구성된 식. 예를 들어, `2x`, `3y`, `-3`은 각각의 항이다.
- **상수항(Constant Term)**: 여러 항 중에서 숫자만으로 이루어진 항.
- **계수(Coefficient)**: 상수와 변수의 곱으로 구성된 항에서, 변수와 곱해진 상수 부분.
- **단항식과 다항식**:
    - **단항식(Monomial)**: 항이 하나로만 이루어진 식.
    - **다항식(Polynomial)**: 두 개 이상의 항의 합으로 이루어진 식. (예: `2x + 3y - 3`)
- **차수(Degree)**: 각 항에서 변수가 곱해진 횟수. 상수항의 차수는 0이다.
    - **일차식**: 다항식의 최대 차수가 1인 식.
    - **이차식**: 다항식의 최대 차수가 2인 식.

**예시 분석: 다항식** `**5x² + 3x - 6y + 4**`

- **항**: `5x²`, `3x`, `-6y`, `4`의 네 개로 구성된 다항식이다.
- **상수항**: `4`
- **차수**:
    - `5x²`의 차수는 2이다.
    - `3x`의 차수는 1이다.
    - `-6y`의 차수는 1이다.
    - `4`의 차수는 0이다.
    - 이 다항식은 x 기준으로는 2차, y 기준으로는 1차이다.

## 3. 방정식과 부등식

수식은 등호(=)를 사용하는 방정식과 부등호(<, ≤, >, ≥)를 사용하는 부등식으로 구분된다.

### 3.1. 방정식의 원리

- **등식(Equality)**: 등호(=)를 기준으로 좌변(left side)과 우변(right side)이 서로 같음을 나타내는 식. 등식은 참일 수도 있고 거짓일 수도 있다.
- **방정식(Equation)**: 미지수(예: x)를 포함하며, 그 미지수의 값에 따라 참이 되기도 하고 거짓이 되기도 하는 등식. 방정식이 성립하려면 반드시 등호와 미지수가 포함되어야 한다.
    - **해 또는 근(Solution/Root)**: 방정식을 참으로 만드는 미지수의 값.
- **방정식의 종류**:
    - **일차방정식**: 최고 차수가 1인 방정식. (예: `x + 1 = 3`)
    - **이차방정식**: 최고 차수가 2인 방정식. (형태: `ax² + bx + c = 0`, 단 `a ≠ 0`)
    - **항등식(Identity)**: 미지수에 어떤 값을 대입하더라도 항상 참이 되는 등식.
    - **연립방정식(System of Equations)**: 여러 개의 미지수를 포함하는 방정식들의 묶음. 해를 구하기 위해서는 일반적으로 미지수의 개수만큼 독립적인 방정식이 필요하다.

#### 방정식과 항등식의 비교

|   |   |   |
|---|---|---|
|구분|방정식 (Equation)|항등식 (Identity)|
|**참인 조건**|미지수가 특정한 값을 가질 때만 참|미지수 값에 관계없이 항상 참|
|**좌변과 우변**|일반적으로 좌변 ≠ 우변|좌변 = 우변|

### 3.2. 부등식의 원리

- **부등식(Inequality)**: 등호 대신 부등호(<, ≤, >, ≥)를 사용하여 수나 식의 대소 관계를 나타내는 식.
    - **절대부등식**: 모든 실수 값에 대해 항상 성립하는 부등식. (예: `x² + 5 ≥ 0`)
    - **조건부등식**: 특정 실수 값에 대해서만 성립하는 부등식. (예: `x + 2 ≤ 7`)
- **부등식의 핵심 성질**:
    - 양변에 같은 수를 더하거나 빼도 부등호의 방향은 바뀌지 않는다.
    - 양변에 **음수**를 곱하거나 나누면 부등호의 방향이 **반대로 바뀐다**.
