
## 요약 (Executive Summary)

본 문서는 데이터 과학과 인공지능 분야의 기반이 되는 행렬 변환의 핵심 개념을 종합적으로 설명한다. 주요 내용은 벡터의 기본 연산(덧셈, 뺄셈, 곱셈), 벡터 간의 거리 및 유사도 측정 방법, 그리고 행렬을 이용한 벡터 공간의 변환 원리로 구성된다.

핵심적으로 벡터의 내적은 인공지능에서 개체 간 유사도를 측정하는 데 활용되며, 외적은 3차원 공간에서의 연산에 사용된다. 또한, 유클리드 거리, 맨해튼 거리, 코사인 유사도는 추천 시스템이나 텍스트 분석과 같은 분야에서 데이터 간의 관계를 정량화하는 중요한 도구로 기능한다. 행렬 변환의 개념은 3D 그래픽에서의 벡터 이동 및 회전부터 선형 변환, 역행렬을 이용한 선형 방정식의 해 계산에 이르기까지 폭넓게 응용된다. 본 자료는 각 개념에 대한 수학적 정의, 주요 성질, 그리고 `Numpy`, `SciPy` 등 파이썬 라이브러리를 활용한 구현 방법을 함께 제시하여 이론과 실제 적용을 연결한다.

## 1. 벡터의 기본 연산

벡터의 연산은 데이터 분석 및 인공지능 모델링의 기초를 형성한다. 벡터의 덧셈, 뺄셈, 곱셈(내적, 외적)은 각각 고유한 수학적 성질과 기하학적 의미를 가진다.

### 가. 벡터의 덧셈과 뺄셈

- **벡터 덧셈**: 두 벡터의 각 원소를 더하는 연산이다. 기하학적으로는 '평행사변형 법칙' 또는 '삼각형 법칙'으로 표현할 수 있다.
    - **성질**:
        - **교환 법칙**: `𝐴 + 𝐵 = 𝐵 + 𝐴`
        - **결합 법칙**: `(𝐴 + 𝐵) + 𝐶 = 𝐴 + (𝐵 + 𝐶)`
        - **항등원 존재**: `𝐴 + 0 = 𝐴`
        - **역원 존재**: `𝐴 + (−𝐴) = 0`
- **벡터 뺄셈**: 한 벡터에 다른 벡터의 음수 벡터를 더하는 것과 동일하며(`𝑎 − 𝑏 = 𝑎 + (−𝑏)`), 주로 두 점 사이의 거리를 계산하는 데 사용된다.

### 나. 벡터의 곱셈

벡터의 곱셈은 내적(Dot Product)과 외적(Cross Product)으로 구분되며, 연산 결과와 그 의미가 상이하다.

#### 1) 벡터 내적 (Dot Product / Inner Product)

- **정의**: 두 벡터의 각 원소를 곱한 후 모두 더하는 연산으로, 결과값은 스칼라(Scalar)이다.
    - **수학적 표현**: `𝑥 ∙ 𝑦 = 𝑋ᵀ𝑌 = 𝑥₁𝑦₁ + 𝑥₂𝑦₂ + ⋯ + 𝑥ₙ𝑦ₙ`
    - **조건**: 연산을 위해서는 두 벡터의 차원이 같아야 하며, 앞 벡터는 행 벡터, 뒤 벡터는 열 벡터 형식이어야 한다.
- **기하학적 의미**: `𝑎 ∙ 𝑏 = |𝑎| |𝑏| cos θ`로 표현되며, 한 벡터를 다른 벡터에 정사영(projection)시킨 크기와 관련이 있다.
- **주요 성질**:
    - **교환 법칙**: `𝑎 ∙ 𝑏 = 𝑏 ∙ 𝑎`
    - **분배 법칙**: `𝑎 ∙ (𝑏 + 𝑐) = 𝑎 ∙ 𝑏 + 𝑎 ∙ 𝑐`
- **활용**: 인공지능 분야에서 개체 간의 **유사도(Similarity) 또는 비유사도(Dissimilarity)**를 측정하는 데 핵심적으로 사용된다.
- **파이썬 구현**: `numpy.dot()` 함수를 사용한다.

#### 2) 벡터 외적 (Cross Product / Outer Product)

- **정의**: 3차원 공간에 존재하는 두 벡터 간의 연산으로, 결과값은 벡터이다.
- **결과 벡터의 특징**:
    - **방향**: 연산에 사용된 두 벡터(`𝑎`, `𝑏`)에 동시에 수직인 방향을 가진다.
    - **크기**: 두 벡터(`𝑎`, `𝑏`)를 변으로 하는 평행사변형의 넓이와 같다.
- **계산 공식**: `𝑎 = (𝑎₁, 𝑎₂, 𝑎₃)`, `𝑏 = (𝑏₁, 𝑏₂, 𝑏₃)`일 때, `𝑎 × 𝑏 = (𝑎₂𝑏₃ − 𝑎₃𝑏₂, 𝑎₃𝑏₁ − 𝑎₁𝑏₂, 𝑎₁𝑏₂ − 𝑎₂𝑏₁)`
- **주요 성질**:
    - **반대칭성**: `𝑎 × 𝑏 = −(𝑏 × 𝑎)` (교환 법칙 성립 안 함)
    - **분배 법칙**: `𝑎 × (𝑏 + 𝑐) = (𝑎 × 𝑏) + (𝑎 × 𝑐)`
- **파이썬 구현**: `numpy.cross()` 함수를 사용한다.

## 2. 벡터 거리와 유사도 측정

벡터 공간에서 두 데이터 포인트(벡터) 간의 거리는 그들의 유사도를 나타내는 척도로 사용된다. 거리가 가까우면 유사도가 높고, 거리가 멀면 유사도가 낮다고 해석한다. 이 개념은 추천 시스템, 텍스트 분석 등 다양한 분야에서 활용된다.

- **유클리드 거리 (Euclidean Distance)**: 두 벡터 간의 가장 짧은 직선거리를 의미한다.
    - `파이썬 구현`: `scipy.euclidean()` 함수 사용.
- **맨해튼 거리 (Manhattan Distance)**: 사각형 격자 형태의 경로를 따라 이동할 때의 최단 거리를 나타낸다.
    - `파이썬 구현`: `math` 라이브러리의 기본 연산으로 계산 가능.
- **코사인 유사도 (Cosine Similarity)**: 두 벡터가 이루는 각도를 이용하여 유사도를 측정한다. 각도가 작을수록(0에 가까울수록) 유사도가 높다(1에 가깝다).
    - **수학적 표현**: `cos θ = (𝐴∙𝐵) / (|𝐴| |𝐵|)`
    - `파이썬 구현`: `numpy.norm()` 함수 등을 이용하여 계산.

|   |   |   |
|---|---|---|
|측정 방식|정의|주요 활용 분야|
|**유클리드 거리**|두 벡터 간의 직선 거리|클러스터링, 일반적인 거리 기반 분석|
|**맨해튼 거리**|격자 경로상의 최단 거리|이미지 처리, 경로 탐색|
|**코사인 유사도**|두 벡터 사이의 각도를 이용한 유사도|추천 시스템, 문서(텍스트) 유사도 분석|

## 3. 행렬 변환의 원리

행렬 변환은 특정 벡터를 다른 벡터로 옮기는 함수(Function)의 개념을 확장한 것이다. 이는 벡터 공간을 다른 벡터 공간으로 변환하는 과정이며, 다양한 수학적 도구와 개념을 포함한다.

### 가. 함수와 벡터 변환

- **함수**: 정의역(Domain)과 공역(Co-domain)이라는 두 집합 사이의 대응 관계.
- **벡터 변환**: 집합의 원소를 벡터로 간주하여, 한 벡터 공간(`Rⁿ`)에서 다른 벡터 공간(`Rᵐ`)으로 벡터를 변환시키는 함수. 3D 게임 등에서 벡터의 회전 및 이동에 사용된다.

### 나. 선형 변환 (Linear Transformation)

- **정의**: 벡터의 덧셈과 스칼라 곱 연산(선형 결합)을 보존하는 벡터 공간 사이의 함수.
- **조건**:
    1. **합의 법칙**: `𝑇(𝑎 + 𝑏) = 𝑇(𝑎) + 𝑇(𝑏)`
    2. **스칼라 곱의 법칙**: `𝑇(𝑐𝑎) = 𝑐𝑇(𝑎)`

### 다. 역함수와 역행렬

#### 1) 역함수 (Inverse Function)

- **정의**: 변수 `x`와 함수값 `y`를 서로 뒤바꾸어 얻는 함수. 원래 함수가 **일대일 대응**일 때만 존재한다.
- **성질**:
    - `(𝑓⁻¹)⁻¹ = 𝑓`
    - `𝑓⁻¹ ∘ 𝑓`는 정의역 `X`에서의 항등함수이다.
    - `𝑓 ∘ 𝑓⁻¹`는 공역 `Y`에서의 항등함수이다.

#### 2) 역행렬 (Inverse Matrix)

- **정의**: 어떤 행렬 `A`와 곱했을 때 단위행렬 `I`가 나오는 행렬. `𝐴𝐵 = 𝐵𝐴 = 𝐼`일 때 `B`는 `A`의 역행렬(`A⁻¹`)이다.
- **존재 조건**: 정사각행렬의 행렬식(Determinant)이 0이 아니어야 한다.
    - 2차 정사각행렬 `A`의 행렬식: `ad - bc`
- **주요 성질**:
    - `(𝐴⁻¹)⁻¹ = 𝐴`
    - `(𝐴𝐵)⁻¹ = 𝐵⁻¹𝐴⁻¹`
- **활용**: 선형 방정식의 해를 구할 때 매우 유용하다 (`𝐴𝑋 = 𝐵` ⇔ `𝑋 = 𝐴⁻¹𝐵`).
- **파이썬 구현**: `numpy.linalg.inv()` 함수를 사용한다.

### 라. 전치행렬 (Transposed Matrix)

- **정의**: 행렬의 행과 열을 서로 맞바꾼 행렬 (`𝑚 × 𝑛` → `𝑛 × 𝑚`). `𝐴ᵀ`로 표기한다.
- **주요 성질**:
    - `(𝐴ᵀ)ᵀ = 𝐴`
    - `(𝐴 + 𝐵)ᵀ = 𝐴ᵀ + 𝐵ᵀ`
    - `(𝐴𝐵)ᵀ = 𝐵ᵀ𝐴ᵀ`
- **전치행렬과 역행렬의 관계**: 가역성을 가지는 행렬의 전치행렬 또한 가역성을 가지며, `(𝐴ᵀ)⁻¹ = (𝐴⁻¹)ᵀ`가 성립한다.
- **파이썬 구현**: `a.T` 또는 `numpy.transpose(a)`를 사용한다.
- **행렬식 (Determinant)**: 행과 열의 개수가 같은 정사각행렬에 특정 수를 대응시키는 함수로, `det(A)`로 표기한다. 역행렬의 존재 여부를 판단하는 데 사용된다.
