
## Executive Summary

본 문서는 확률론의 네 가지 핵심 주제인 집합, 확률, 조건부 확률, 그리고 확률 변수와 확률 분포에 대한 종합적인 분석을 제공한다. 이 개념들은 데이터 과학과 인공지능의 근간을 이루는 필수적인 요소이다.

1. **집합(Sets):** 확률론의 언어를 구성하는 기본 단위로, 구별 가능한 객체의 모임으로 정의된다. 합집합, 교집합, 차집합 등의 연산을 통해 사건(event) 간의 관계를 명확히 기술할 수 있으며, 파이썬의 `set`과 `frozenset` 자료형을 통해 컴퓨터 과학적으로 표현 및 조작이 가능하다.
2. **확률(Probability):** 특정 사건이 발생할 가능성을 0과 1 사이의 실수 값으로 나타낸 척도이다. 콜모고로프의 세 가지 공리(음이 아닌 실수, 전체 사건 확률은 1, 배반사건의 합집합 확률)를 기반으로 수학적으로 정의된다. 확률은 반복 실험의 결과로 보는 '빈도주의' 관점과, 주어진 정보를 바탕으로 한 주장의 신뢰도로 보는 '베이즈주의' 관점으로 해석될 수 있다.
3. **조건부 확률(Conditional Probability):** 한 사건이 발생했다는 조건 하에 다른 사건이 발생할 확률을 의미하며, 새로운 정보가 주어졌을 때 확률적 추론을 갱신하는 핵심 메커니즘이다. 이를 바탕으로 도출된 **베이즈 정리(Bayes' Theorem)**는 사전 확률(prior)을 우도(likelihood)와 증거(evidence)를 통해 사후 확률(posterior)로 업데이트하는 과정을 명시하며, 이는 분류 문제와 같은 기계 학습 분야에서 광범위하게 활용된다.
4. **확률 변수와 확률 분포(Random Variables and Distributions):** 실험의 결과를 실수 값에 대응시키는 함수를 확률 변수라 하며, 이는 이산형과 연속형으로 나뉜다. 확률 분포는 확률 변수가 가질 수 있는 값들과 그 값에 해당하는 확률들을 총체적으로 나타내는 것으로, 확률질량함수(PMF), 누적분포함수(CDF), 확률밀도함수(PDF) 등을 통해 수학적으로 정밀하게 기술된다.

## 1. 집합: 확률론의 기본 구성 요소

집합은 확률론의 개념을 정의하고 사건 간의 관계를 설명하는 데 사용되는 근본적인 도구이다.

### 핵심 정의

- **집합(Set):** 구별 가능한 객체의 모임.
- **원소(Element):** 집합에 포함된 개별 객체. 원소 `x`가 집합 `A`에 포함되는 관계는 `𝑥 ∈ 𝐴`로 표현한다.
- **집합의 크기(Cardinality):** 집합이 가지는 원소의 수. `|𝐴|` 또는 `card(A)`로 표기한다.
- **전체집합(Universal Set, Ω):** 논의의 대상이 되는 모든 원소를 포함하는 집합.
- **부분집합(Subset):** 어떤 집합의 원소 중 일부 또는 전부를 포함하는 집합. `𝐴 ⊂ 𝐵`로 표기한다. 원소의 개수가 N개인 집합은 총 `2^N`개의 부분집합을 가진다.
- **공집합(Null Set, Ø):** 원소를 하나도 포함하지 않은 집합. 크기는 0이다.

### 집합 연산 및 법칙

|   |   |   |   |
|---|---|---|---|
|연산 유형|설명|기호|파이썬 구현|
|**합집합**|두 집합의 모든 원소를 포함하는 집합|`𝐴 ∪ 𝐵`|`A.union(B)` 또는 `A \| B`|
|**교집합**|두 집합에 공통으로 속하는 원소로 이루어진 집합|`𝐴 ∩ 𝐵`|`A.intersection(B)` 또는 `A & B`|
|**차집합**|집합 A에는 속하지만 집합 B에는 속하지 않는 원소들의 집합|`𝐴 − 𝐵`|`A.difference(B)` 또는 `A - B`|
|**여집합**|전체집합 Ω 중 부분집합 A에 속하지 않는 원소들의 집합|`𝐴^𝑐`|`Ω - A`|

**분배 법칙:** 집합의 합집합과 교집합 연산은 다음과 같은 분배 법칙을 따른다.

- `𝐴 ∪ (𝐵 ∩ 𝐶) = (𝐴 ∪ 𝐵) ∩ (𝐴 ∪ 𝐶)`
- `𝐴 ∩ (𝐵 ∪ 𝐶) = (𝐴 ∩ 𝐵) ∪ (𝐴 ∩ 𝐶)`

### 파이썬을 이용한 구현

파이썬에서는 집합을 표현하기 위해 두 가지 주요 자료형을 제공한다.

- `**set**`**:** 내용을 변경할 수 있는(mutable) 집합.
- `**frozenset**`**:** 내용을 변경할 수 없는(immutable) 집합으로, 딕셔너리의 키나 다른 `set`의 원소가 될 수 있다.
- **참고:** 무한집합은 파이썬의 `set` 또는 `frozenset`으로 표현할 수 없다.

## 2. 확률의 정의와 관점

확률은 불확실한 현상을 수학적으로 모델링하는 핵심 개념으로, 사건 발생의 가능성을 정량화한다.

### 기본 개념: 표본 공간, 사건, 확률

- **실험(Experiment):** 동일 조건에서 반복 가능하며 결과가 우연에 의해 결정되는 관찰.
- **확률 표본(Sample):** 실험에서 발생할 수 있는 하나의 현상 또는 결과.
- **표본 공간(Sample Space, Ω):** 가능한 모든 확률 표본의 집합.
- **사건(Event):** 표본 공간의 부분집합. 즉, 하나 이상의 표본으로 구성된 집합.

**확률의 정의:** 어떤 실험에서 사건 A가 일어날 가능성을 수로 나타낸 것으로, 표본이 아닌 사건을 입력으로 받아 0과 1 사이의 실수 값을 출력하는 함수이다. 수학적으로는 `P(A) = n(A) / n(S)`로 표현될 수 있으며, 이는 `A가 일어날 경우의 수 / 모든 경우의 수`를 의미한다.

### 콜모고로프의 공리적 정의

확률은 다음 세 가지 공리를 반드시 만족해야 한다.

1. **비음수성:** 모든 사건 A에 대해 확률은 0 또는 양수이다 (`P(A) ≥ 0`).
2. **정규성:** 표본 공간(전체 사건)의 확률은 1이다 (`P(Ω) = 1`).
3. **가산성:** 서로 공통 원소가 없는 두 사건(배반사건) A와 B의 합집합에 대한 확률은 각 사건의 확률의 합과 같다 (`A ∩ B = Ø` 이면 `P(A ∪ B) = P(A) + P(B)`).

### 확률에 대한 두 가지 관점

|   |   |   |
|---|---|---|
|관점|설명|예시|
|**빈도주의(Frequentist)**|확률을 무수히 많은 반복 실험을 통해 얻어지는 객관적인 빈도로 간주한다.|"동전의 앞면이 나올 확률이 0.5다." (수없이 던졌을 때 앞면이 나오는 비율이 0.5에 수렴)|
|**베이즈주의(Bayesian)**|확률을 실험 전 주어진 정보를 바탕으로 한 특정 가설이나 주장에 대한 '신뢰도'로 해석한다.|"새가 날 수 있는 가능성은 95%다." (관찰과 지식에 기반한 믿음의 정도)|

### 확률의 주요 성질

- **공집합의 확률:** `P(Ø) = 0`
- **여집합의 확률:** `P(A^c) = 1 - P(A)`
- **덧셈 규칙 (포함-배제 원리):** `P(A ∪ B) = P(A) + P(B) - P(A ∩ B)`
- **전체 확률의 법칙:** 표본 공간을 분할하는 배반사건들 `C_i`에 대해, `P(A) = Σ P(A ∩ C_i)`가 성립한다.

## 3. 조건부 확률과 베이즈 정리

조건부 확률은 추가적인 정보가 주어졌을 때 확률을 어떻게 갱신해야 하는지에 대한 수학적 틀을 제공한다.

### 결합, 주변, 조건부 확률

- **결합 확률(Joint Probability):** 두 사건 A와 B가 동시에 발생할 확률. `P(A ∩ B)` 또는 `P(A, B)`로 표기.
- **주변 확률(Marginal Probability):** 다른 사건에 관계없이 개별 사건 A 또는 B가 발생할 확률. `P(A)` 또는 `P(B)`.
- **조건부 확률(Conditional Probability):** 사건 B가 발생했다는 가정 하에 사건 A가 발생할 확률. 다음과 같이 정의된다: `P(A|B) = P(A ∩ B) / P(B)`

### 독립 사건과 종속 사건

- **독립 사건(Independent Events):** 한 사건의 발생이 다른 사건의 발생 확률에 영향을 주지 않는 경우. 이 경우 `P(A|B) = P(A)` 이며, 필요충분조건은 `P(A ∩ B) = P(A)P(B)`이다.
- **종속 사건(Dependent Events):** 한 사건의 발생이 다른 사건의 발생 확률에 영향을 주는 경우.

### 베이즈 정리의 원리 및 활용

베이즈 정리는 조건부 확률의 정의로부터 유도되며, 새로운 정보를 바탕으로 기존의 믿음(확률)을 갱신하는 강력한 도구이다.

`P(A|B) = [P(B|A) * P(A)] / P(B)`

- `**P(A|B)**` **(사후 확률, Posterior):** 사건 B가 관측된 후의 사건 A에 대한 확률. (갱신된 믿음)
- `**P(B|A)**` **(우도/가능도, Likelihood):** 사건 A가 사실일 때 사건 B가 관측될 확률.
- `**P(A)**` **(사전 확률, Prior):** 사건 B를 관측하기 전의 사건 A에 대한 초기 확률. (기존의 믿음)
- `**P(B)**` **(증거, Evidence):** 사건 B가 발생할 전체 확률. 정규화 상수로 작용한다.

**베이즈 정리의 확장:**

1. **다중 분류 문제:** 표본 공간을 분할하는 여러 사건 `A_i`에 대해 특정 `A_i`의 사후 확률을 계산하는 데 사용될 수 있다.
2. **순차적 갱신:** 사건 B에 의해 갱신된 확률은, 또 다른 사건 C가 발생했을 때 다시 한번 갱신될 수 있다.

## 4. 확률 변수와 확률 분포

확률 변수와 분포는 실험의 결과를 수치적으로 다루고, 그 결과들이 나타나는 패턴을 수학적으로 모델링하는 데 사용된다.

### 확률 변수의 정의 및 종류

- **확률 변수(Random Variable):** 실험 결과에 따라 표본 공간의 각 원소에 하나의 실수 값을 대응시키는 함수.
- **이산형 확률 변수(Discrete Random Variable):** 확률 변수가 가질 수 있는 값이 유한하거나 셀 수 있는 경우 (예: 주사위 두 개의 눈의 합).
- **연속형 확률 변수(Continuous Random Variable):** 확률 변수가 특정 구간 내의 모든 실수 값을 가질 수 있는 경우 (예: 사람의 키).

### 확률 분포 함수

확률 분포는 확률 변수가 취하는 값과 그에 대응하는 확률들의 분포를 나타내는 것으로, 다음 함수들을 통해 표현된다.

|   |   |   |   |   |
|---|---|---|---|---|
|함수 종류|약어|대상 변수|설명|수식|
|**확률질량함수**|PMF|이산형|특정 값 `x`가 나타날 정확한 확률을 나타냄.|`f(x) = P[X = x]`|
|**누적분포함수**|CDF|이산형/연속형|확률 변수가 특정 값 `x`보다 작거나 같을 확률을 나타냄. 연속형 변수에서 구간 확률 `P(a < X ≤ b)`는 `F(b) - F(a)`로 계산됨.|`F(x) = P[X ≤ x]`|
|**확률밀도함수**|PDF|연속형|누적분포함수(CDF)의 도함수. 특정 값에서의 확률은 0이지만, 이 함수의 값은 해당 지점에서의 상대적인 확률 밀도를 나타냄. 특정 구간의 확률은 PDF를 해당 구간에 대해 적분하여 구함.|`f(x) = dF(x) / dx`|
